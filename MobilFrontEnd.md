# Mobil Front-End — Stocker

> Bu projede ayrı bir native mobil uygulama (iOS/Android) geliştirilmemiştir.
> Web uygulaması **mobil uyumlu (responsive)** olarak tasarlanmıştır; modern mobil tarayıcılarda
> tam işlevsellikle çalışır.

---

## Mobil Uyumluluk

Web uygulaması ([stocker-olive.vercel.app](https://stocker-olive.vercel.app)) aşağıdaki mobil
özelliklerle donatılmıştır:

| Özellik | Detay |
|---------|-------|
| Viewport meta etiketi | `width=device-width, initial-scale=1.0` |
| Esnek ızgara | CSS Grid — ekran genişliğine göre kolon sayısı değişir |
| Buton boyutları | Parmakla rahat tıklanabilecek büyüklükte |
| Dokunmatik menü | Katalog seçici portal ile taşan içerik sorunsuz açılır |
| Tema rengi | `#ff5733` (tarayıcı adres çubuğu rengi) |

---

## Planlanan Geliştirmeler

Gelecekte React Native veya Flutter ile native mobil uygulama eklenebilir.
Mevcut REST API (`https://stocker-vou5.vercel.app`) değişmeden kullanılabilir.
